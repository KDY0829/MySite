## 📌 scripts.js 기능 요약

이 프로젝트의 scripts.js 파일에는 다음과 같은 주요 기능들이 포함되어 있습니다.

## 1. ⏰ 시간대별 인사말 변경

현재 시간을 기반으로

- 새벽
- 아침
- 오후
- 저녁
  시간대에 따라 홈 화면의 인사말이 자동으로 바뀝니다.

## 2. 🌙 다크모드 / 라이트모드 전환

다크모드 토글 버튼을 클릭하면

- 다크모드 → 라이트모드
- 라이트모드 → 다크모드
  로 화면 색상 테마가 전환됩니다.

## 3. 🧭 메뉴 클릭 시 부드러운 스크롤 이동 (Smooth Scroll)

헤더 메뉴를 클릭하면 해당 섹션으로 부드럽게 스크롤링 되도록 처리됩니다.
(갑자기 점프하지 않음)

## 4. 🎛 포트폴리오 카테고리 필터

포트폴리오 섹션에서

- ALL
- Front-end
- Mobile
- Back-end
  버튼을 클릭하면 해당 카테고리 프로젝트만 표시되고
  나머지는 숨겨집니다.

## 5. 🎬 스크롤 시 섹션 등장 애니메이션

사용자가 페이지를 스크롤할 때 각 섹션이 화면에 들어오는 순간

- 페이드 인(서서히 나타나기)
- 슬라이드 인(아래에서 위로 부드럽게 이동)
  애니메이션 효과가 적용됩니다.

## 6. 🧩 초기화 함수 (applyAll)

페이지가 로드되면 자동으로 위 기능들을 한 번에 실행하는 초기화 함수가 포함되어 있어
전체 스크립트 기능이 정리된 구조로 실행됩니다.
